My Projects

AirportProject (Scala)

Airport Management Application

A Scala application that analyzes three CSV files (countries, airports, runways) and offers search by code or approximate name, as well as report generation.

  • Custom CSV parsing (no external library used)
  • Fuzzy matching via me.xdrop.fuzzywuzzy
  • Reports (Top 10 countries with most/least airports, etc.)
  • JavaFX Interface and a CLI mode
  • Scala Futures for parallel processing

Run with sbt clean compile run (GUI by default) or modify mainClass for CLI mode. Includes tests via sbt test.

Unicold Project

Refrigeration Product Management Platform

Unicold is a platform that facilitates the management of refrigeration products for companies specializing in cold storage, shelving, and refrigeration equipment. The application consists of two parts:

  • Unicold Front: Customer interface for browsing products and placing orders.
  • Unicold Admin: Admin panel for managing users, products, and orders.

The platform makes it easier for customers to navigate, place orders, and allows administrators to efficiently manage inventory and sales.
