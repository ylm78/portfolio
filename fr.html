Mes Projets

AirportProject (Scala)

Application de gestion d’aéroports

Application Scala qui analyse trois fichiers CSV (pays, aéroports, pistes) et propose une recherche par code ou nom approximatif, ainsi que de la génération de rapports.

  • Parsing CSV personnalisé (pas de lib externe)
  • Fuzzy matching via me.xdrop.fuzzywuzzy
  • Rapports (Top 10 pays avec plus/moins d’aéroports, etc.)
  • Interface JavaFX et un mode CLI
  • Futures Scala pour le traitement parallèle

Se lance avec sbt clean compile run (GUI par défaut) ou en modifiant mainClass pour le mode CLI. Contient des tests via sbt test.

Unicold Project

Plateforme de gestion de produits frigorifiques

Unicold est une plateforme permettant la gestion des produits d’une entreprise spécialisée dans les chambres froides, rayonnages et équipements frigorifiques. L’application se divise en deux parties :

  • Unicold Front : Interface utilisateur pour les clients et la gestion des commandes.
  • Unicold Admin : Interface d’administration pour gérer les utilisateurs, produits et commandes.

L’application facilite la navigation des clients, la passation de commandes et la gestion efficace des stocks par les administrateurs.

Classification de Brevets

Machine Learning & Propriété Intellectuelle

Ce projet utilise des algorithmes de machine learning pour analyser les textes des brevets et les classifier dans les catégories appropriées de la Classification Internationale des Brevets (CIB). Une classification précise est essentielle pour la recherche de brevets, l’analyse des tendances technologiques et la gestion de la propriété intellectuelle.

  • Traitement et vectorisation des textes brevets
  • Utilisation de modèles NLP avancés (TF-IDF, Word Embeddings, Transformers)
  • Apprentissage supervisé avec des algorithmes de classification
  • Évaluation et optimisation des performances des modèles
